计算两个经纬度之间的距离

本文使用的是距离公式是有google地图提供的。

公式说明

对上面的公式解释如下：
1.Lat1 Lng1 表示A点经纬度，Lat2 Lng2 表示B点经纬度；
2.a=Lat1 – Lat2 为两点纬度之差 b=Lng1 -Lng2 为两点经度之差；
3.6378.137为地球半径，单位为千米；
计算出来的结果单位为千米，若将半径改为米为单位则计算的结果单位为米。
计算精度与谷歌地图的距离精度差不多，相差范围在0.2米以下。
